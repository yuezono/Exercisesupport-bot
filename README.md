# 運動応援bot（ポートフォリオ）
## 機能
##### 時間測定
開始と終了のキーワードにて時間を計測。  
1日のトータル時間についても計測可能。
##### 動画
『筋トレ』と入力するとランダムで動画のURLが送られてくる。
##### 応援メッセージ
『筋トレ』、『開始・終了』以外の文字を送った際に、ランダムで励ましのメッセージが送られてくる。
